# HEARTBEAT.md - 主动检查清单

## 每次心跳（30分钟）必须执行

**1. 系统状态自检**
- Gateway 进程是否正常（port 18789）
- 今日是否有异常日志（Error/WARN）
- 磁盘空间是否充足

**2. 记忆维护**
- 检查 memory/YYYY-MM-DD.md 是否存在
- 记录本次心跳时间戳和状态

**3. 用户情境感知**
- 距离上次交互是否 >4 小时
- 是否有需要主动推送的信息（如任务完成通知）

## 主动推送触发条件

满足任一即推送飞书：
- 系统异常（进程崩/磁盘满/配置错误）
- 发现可优化项（skill更新/配置加固建议）
- 任务执行完毕（结果汇报）
- 距上次交互 >6 小时且有新发现
- 安全风险（groupPolicy变更/新插件安装）

## 不推送场景

- 深夜（23:00-08:00），除非紧急
- 正常状态下的常规心跳
- 距上次推送 <30 分钟

## 执行日志

心跳执行后记录到 memory/YYYY-MM-DD.md
