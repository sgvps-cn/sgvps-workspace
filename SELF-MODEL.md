## 最近学到的教训（2026-03-31）

1. **SEO prompt未禁止H1** → API按字面生成，文章内容出现多个H1
2. **DOTALL regex `\n`** → `[\r\n\\]*` 只匹配字面不匹配真实换行
3. **subprocess fork** → 在cron下subprocess.run会fork，可能导致双重执行
4. **cron环境PATH受限** → 所有脚本命令用绝对路径
5. **没有监控等于没运行** → evolver停止日志2小时才被发现
